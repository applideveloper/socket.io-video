socket.io-video
===============

FaceTime Camera share using socket.io 

Install
----------

```shell
$ git clone https://github.com/yosuke-furukawa/socket.io-video.git
$ cd socket.io-video
$ npm install
$ node app.js
```

Lauch your webcam and use chrome browser.
